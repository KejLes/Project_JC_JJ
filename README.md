# Project_JC_JJ
Hecho por Joaquin Canales y Jean Juarez
Este proyecto será una GESTIÓN DE COLECCIÓN DE VIDEOJUEGOS, ...

Se hará programación orientada a objetos, donde tendremos una gran variedad de videojuegos, los cuales seran clasificadas y tendrán diferentes atributos gracias al uso de la herencia. En el programa se podrá ver la lista de videojuegos, buscar el videojuego, ver sus características,
ver si está en venta, y si lo está, ver dónde lo venden. Algunas características del videojuego pueden ser, el género, las plataformas donde se puede jugar, la desarrolladora, incluso la franquicia a la que pertenece, año de publicación, plataformas disponiles donde jugar.
¿Quizás añadir una opción para comprar si es que está en venta? Quizás que se incluya el link de la página del videojuego?
